Deep Neural Network Model

Post preprocessing we knew a multiple linear regression was probably insufficient so we explored the TensorFlow library for a layered solution. Their gas mileage prediction tutorial had some good recommendations that fit our data format. (TensorFlow - Gas Mileage Prediction Regression)
We still used Sklearn's Train_Test_Split feature divide and test the data. A Normalization layer was added based on the X_train data instead of a scaler to account for the binary coding and to help the model converge. The 'Relative' sentiment value was selected as the targeted label.

Results

The finalized model was trained on stock data from January - October 2017 with the goal of trying to accurately predict the sentiment value based on the known November stock data for the major car makers. In testing the model returned:

  • Epochs: 1200
  • Mean Squared Error (MSE) loss value of 1.0960
  • Mean Absolute Percentage Error (MAPE) of 0.4899
